# BGMI BOT v9.0

Telegram bot for BGMI match testing.

## Commands

### Users
- `/attack <ip:port> <sec>` - Attack (10-300s)
- `/stop` - Stop attack
- `/myplan` - Your plan
- `/plans` - Buy plan
- `/redeem <code>` - Activate

### Admin (@ayan_vip_admin)
- `/add <id> <days>` - Add plan
- `/remove <id>` - Remove plan
- `/users` - List users
- `/broadcast <msg>` - Message all
- `/stats` - Bot stats
- `/stopall` - Stop all
- `/gencode <hours>` - Generate code
